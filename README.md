# Linux Kiosk Infrastructure

Ubuntu Server based digital signage kiosk infrastructure using Rise Vision Player.

## Features

- Ubuntu Server kiosk deployment
- Rise Vision Player integration
- Xorg + Openbox kiosk mode
- SSH hardening
- systemd automation
- Monitor scheduling
- Auto-login configuration
- Deployment documentation

## Technologies

- Ubuntu Server
- Linux
- SSH
- Xorg
- Openbox
- systemd
- Rise Vision
